# Finance Performance Dashboard -Power BI

## Project Overview
This project presents a Finance View dashboard created using Power BI to analyze financial performance. The dashboard provides interactive visualizations that help understand key financial metrics such as revenue, expenses, and profit trends. It enables users to quickly explore financial data and support data-driven decision-making.

## Key Insights
- Overview of total revenue, expenses, and profit
- Monthly financial performance trends
- Profit and expense comparison
- Interactive filters for deeper financial analysis

## Tools Used
- Power BI
- Data Visualization
- Data Analysis
